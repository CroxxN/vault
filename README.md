# A Encryption and hashing library

## Vault is a Encryption and hashing library written in Rust

### Implementation goals:

- Secure Hash Algorithm Family:
  - [ ] SHA0 
  - [ ] SHA1 
  - [ ] SHA2 

- AES Family:
  - [ ] AES-128
  - [ ] AES-192
  - [ ] AES-256

- CRC:
  - [ ] CRC8
  - [ ] CRC16
  - [ ] CRC32
